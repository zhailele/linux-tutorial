# Apollo

Apollo（阿波罗）是携程框架部门研发的分布式配置中心，能够集中化管理应用不同环境、不同集群的配置，配置修改后能够实时推送到应用端，并且具备规范的权限、流程治理等特性，适用于微服务配置管理场景。

官方 Github：https://github.com/ctripcorp/apollo

由于官方示例不能直接使用，所以我 Fork 后，略作修改：https://github.com/dunwu/apollo
